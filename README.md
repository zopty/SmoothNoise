# SmoothNoise

シンプレックスノイズを用いた、連続的な時間変化と滑らかな勾配のノイズです。

beta22a にて動作確認済。

# 導入方法

1. [リリースページ](https://github.com/zopty/SmoothNoise/releases/latest/) から `SmoothNoise.anm2` をダウンロード
2. AviUtl2 の画面に `SmoothNoise.anm2` をドラッグ＆ドロップ
3. `このプラグイン・スクリプトを信頼して使用する` をクリック
4. `加工` に追加されます
